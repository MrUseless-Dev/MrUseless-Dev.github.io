HTML und CSS Basics

HTML File Struktur

<!DOCTYPE html>

Der Dokumententyp (DOCTYPE) gibt an ob es sich zum Beispiel um eine HTML-, XHTML, XML, MathML oder SVG-Datei handelt. <!DOCTYPE html> legt die Document Type Definition (DTD) des Dokuments fest. Durch die DTD wird die Grammatik und Syntax der HTML-Sprache festgelegt.

<head>...</head>

Unter dem HTML head befinden sich Metadaten (title, meta und base), die verschiedene Informationen zur Seite liefern, wie zum Beispiel Beziehungen zu anderen Dateien (style, link, script). Was im head-Tag steht, wird nicht auf der Webseite angezeigt. Ausgenommen davon ist der <title>, welcher im oberen Fensterrahmen des Browsers angezeigt wird.

<body>...</body>

Im body befindet sich der gesamte visuelle Inhalt der Webseite.

HTML-Syntax

			
<!DOCTYPE html>
<html>
<head>
	<title>Das ist ein Titel</title>
</head>
<body>
	<!--STYLE YOUR SITE-->
	</body>
</html>

HTML und CSS verlinken

Für die graphische Gestaltung der HTML Seite wird ein CSS File (stylesheet) verwendet. Dadurch gibt es eine strikte Trennung zwischen Inhalt und Aussehen der Webseite. Das Stylesheet beschreibt wie die verschiedenen Elemente (Farbe, Schriftart, Größe) einer Webseite ausschauen.

HTML-Syntax

			
<link rel="stylesheet" href="<filename>">

Überschriften

Überschriften in HTML sind durch ihr <hx> Tag erkennbar, wobei das x für eine Zahl von 1 bis 6 steht. Durch die Zahlen können sehr einfach unter Überschriften erzeugt werden.

HTML-Syntax

			
<h1>Das ist eine Überschrift 1</h1>
<h2>Das ist eine Überschrift 2</h2>

Output

Das ist eine Überschrift 1

Das ist eine Überschrift 2

Paragrafen

Paragrafen machen genau das was man sich denkt, sie unterteilen die Texte der HTML Seiten in einzelne Paragrafen. Ein Paragrafen startet immer in einer neuen Zeile und wird von Browsern automatisch durch ein New Line getrennt.

HTML-Syntax

			
<p>Das ist ein Paragraph</p>
<p>Hier ist noch ein Paragraph</p>

Output

Das ist ein Paragraph

Hier ist noch ein Paragraph

Bilder einbinden

Auch Bilder können auf Webseiten eingebunden werden.

HTML-Syntax

			
<img src="<filename>" alt="<Alternative Text>"/>

Output

Links zu anderen Websites

Links zu anderen Seiten der eigenen Homepage bzw. externe Links können auf einer Webseite eingebunden werden. Das Attribut target="_blank" dient dabei dazu einen neuen Tab zu öffnen.

Bild als Link

HTML-Syntax

			
<a href="<reference Link>" target="_blank"><img src="<filename>" alt="<Alternative Text>"/></a>

Output

Link zu externer Seite

HTML-Syntax

			
<a href="https://pixabay.com/de/" target="_blank">pixabay</a>

Output

pixabay

(Alle Bilder dieser Webseite stammen von pixabay)
