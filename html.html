	<h1>HTML und CSS Basics</h1>
	<h2>HTML File Struktur</h2>
	<h3><!DOCTYPE html></h3>
	<p>
		Der Dokumententyp (DOCTYPE) gibt an ob es sich zum Beispiel um eine HTML-, XHTML, XML, MathML oder SVG-Datei
		handelt.
		<!DOCTYPE html> legt die Document Type Definition (DTD) des Dokuments fest. Durch die DTD wird die
		Grammatik und Syntax der HTML-Sprache festgelegt.
	</p>
	<h3><head>...</head></h3>
	<p>
		Unter dem HTML head befinden sich Metadaten (title, meta und base), die verschiedene Informationen zur Seite
		liefern,
		wie zum Beispiel Beziehungen zu anderen Dateien (style, link, script). Was im head-Tag steht, wird nicht auf der
		Webseite angezeigt.
		Ausgenommen davon ist der <title>, welcher im oberen Fensterrahmen des Browsers angezeigt wird.
	</p>
	<h3><body>...</body></h3>
	<p>
		Im body befindet sich der gesamte visuelle Inhalt der Webseite.
	</p>
	<h3>HTML-Syntax</h3>
	<div class="syntax">
		<pre>
			<code>
<!DOCTYPE html>
<html>
<head>
	<title>Das ist ein Titel</title>
</head>
<body>
	<!--STYLE YOUR SITE-->
	</body>
</html>
			</code>
		</pre>
	</div>


	<h2>HTML und CSS verlinken</h2>
	<p>
		Für die graphische Gestaltung der HTML Seite wird ein CSS File (stylesheet) verwendet.
		Dadurch gibt es eine strikte Trennung zwischen Inhalt und Aussehen der Webseite.
		Das Stylesheet beschreibt wie die verschiedenen Elemente (Farbe, Schriftart, Größe) einer Webseite ausschauen.
	</p>
	<h3>HTML-Syntax</h3>
	<div class="syntax">

		<pre>
			<code>
<link rel="stylesheet" href="<filename>">
			</code>
		</pre>
	</div>


	<h2>Überschriften</h2>
	<p>
		Überschriften in HTML sind durch ihr <hx> Tag erkennbar, wobei das x für eine Zahl von 1 bis 6 steht.
		Durch die Zahlen können sehr einfach unter Überschriften erzeugt werden.
	</p>
	<h3>HTML-Syntax</h3>
	<div class="syntax">
		<pre>
			<code>
<h1>Das ist eine Überschrift 1</h1>
<h2>Das ist eine Überschrift 2</h2>
			</code>
		</pre>
	</div>
	<h3>Output</h3>
	<div class="output">
		<h1>Das ist eine Überschrift 1</h1>
		<h2>Das ist eine Überschrift 2</h2>
	</div>


	<h2>Paragrafen</h2>
	<p>
		Paragrafen machen genau das was man sich denkt, sie unterteilen die Texte der HTML Seiten in einzelne
		Paragrafen.
		Ein Paragrafen startet immer in einer neuen Zeile und wird von Browsern automatisch durch ein New Line getrennt.
	</p>
	<div class="syntax">
		<h3>HTML-Syntax</h3>
		<pre>
			<code>
<p>Das ist ein Paragraph</p>
<p>Hier ist noch ein Paragraph</p>
			</code>
		</pre>
	</div>
	<h3>Output</h3>
	<div class="output">
		<p>Das ist ein Paragraph</p>
		<p>Hier ist noch ein Paragraph</p>
	</div>


	<h2>Bilder einbinden</h2>
	<p>
		Auch Bilder können auf Webseiten eingebunden werden.
	</p>
	<h3>HTML-Syntax</h3>
	<div class="syntax">
		<pre>
			<code>
<img src="<filename>" alt="<Alternative Text>"/>
			</code>
		</pre>
	</div>
	<h3>Output</h3>
	<div class="output_img">
		<img id="image" src="media/img.jpg" alt="Bild von HTML Code" />
	</div>


	<h2>Links zu anderen Websites</h2>
	<p>
		Links zu anderen Seiten der eigenen Homepage bzw. externe Links können auf einer Webseite eingebunden werden.
		Das Attribut target="_blank" dient dabei dazu einen neuen Tab zu öffnen.
	</p>
	<h3>Bild als Link</h3>
	<h4>HTML-Syntax</h4>
	<div class="syntax">
		<pre>
			<code>
<a href="<reference Link>" target="_blank"><img src="<filename>" alt="<Alternative Text>"/></a>
			</code>
		</pre>
	</div>
	<h4>Output</h4>
	<div class="output_without_Border">
		<a href="https://quickdraw.withgoogle.com/" target="_blank"><img id="clickImage" src="media/click-img.png"
				alt="Bild von HTML Code" /></a>
	</div>
	<h3>Link zu externer Seite</h3>
	<h4>HTML-Syntax</h4>
	<div class="syntax">
		<pre>
			<code>
<a href="https://pixabay.com/de/" target="_blank">pixabay</a>
			</code>
		</pre>
	</div>
	<h4>Output</h4>
	<div class="output">
		<a href="https://pixabay.com/de/" target="_blank">pixabay</a>
		<p>(Alle Bilder dieser Webseite stammen von pixabay)</p>
	</div>


	<h2>Listen</h2>
	<p>
		Listen können sowohl mit Punkten (unordert List) als auch mit Nummerierung (orderet List) dargestellt werden.
	</p>
	<h3>HTML-Syntax</h3>
	<div class="syntax">
		<pre>
			<code>
<ul>
	<li>...</li>
	<li>...</li>
	<li>...</li>
</ul>

<ol>
	<li>...</li>
	<li>...</li>
	<li>...</li>
</ol>
			</code>
		</pre>
	</div>
	<h3>Output</h3>
	<div class="output">
		<p>Liste aller verwendeten HTML Elementen</p>
		<ul>
			<li><h1>...</h1></li>
			<li><h2>...</h2></li>
			<li><h3>...</h3></li>
			<li><p>...</p></li>
			<li><a>...</a></li>
			<li><img></li>
			<li><div>...</div></li>
			<li><pre>...</pre></li>
			<li><code>...</code></li>
			<li><ol>
				<ol>
					<li><li>...</li></li>
					<li><li>...</li></li>
					<li><li>...</li></li>
				</ol>
				</ol>
			</li>
			<li><audio>...</audio></li>
			<li><video>...</video></li>
		</ul>
	</div>
